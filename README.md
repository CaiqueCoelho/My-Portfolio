### How to run

- npm i
- npm run start

### Troubleshooting

- If you are geting any problem with node version or node-gyp problem you can run:
    ```sh
    source ~/.nvm/nvm.sh
    nvm use
    npm i
    npm run start
    ```

- If you are getting an error like `ERR_OSSL_EVP_UNSUPPORTED` use:
    ```sh
    export NODE_OPTIONS=--openssl-legacy-provider
    ```
